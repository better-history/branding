--------------------------------------------------
Псоледние изменения

--------------------------------------------------

Хром - это превосходный браузер, но иногда просмотр истории посещений не удобен. А ведь может быть намного лучше, и "Better History" поможет Вам в этом. См. ниже!

Удаление:
● истории посещений за день
● истории отдельных посещений
● истории в пределах одного домена
● всей истории

Группировка:
● истории каждые 60, 30 или 15 минут
● истории по выбранному домену
● 12/24-часовой форматы времени
● настраиваемый режим группировки

Систематизация:
● фильтрация истории по дням

Поиск:
● по последним нескольким месяцам
● по дате, URL или заголовку страницы
● выделение результатов поиска
● опции контекстного меню для поиска по домену
● опции контекстного меню для поиска по выделенному тексту

Оформление:
● стилизация под интерфейс Хрома
● доступ через кнопку на панели расширений 
● запоминание состояния при закрытии
● поддержка кнопки "Назад"

... со времением будет еще больше.
