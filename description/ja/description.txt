--------------------------------------------------
Recent Changes

--------------------------------------------------

Chrome は素晴らしいブラウザだけど、閲覧履歴の表示には改善の余地があります。Better History でもっと素敵な閲覧履歴を。


閲覧履歴の削除:
● 一日単位での履歴の削除
● 履歴の個別削除
● 同一ドメインの閲覧履歴の一括削除
● 前閲覧履歴の削除

グループ化機能:
● 1時間・30分・15分毎に履歴を自動グループ化
● ドメインごとの閲覧履歴を自動グループ化

Organizing:
● 日付単位でのフィルタリング

検索機能:
● ここ数ヶ月の閲覧履歴の検索
● 日付・URL・ページタイトルから履歴を検索
● 検索語のハイライト表示
● 右クリックオプションでドメイン単位の検索
● 右クリックオプションで選択範囲からの検索

Styling:
● Chrome の UI らしいスタイル
● ツールバーのボタンから履歴を表示可能
● 履歴ページの状態の再現
● 『戻る』ボタンによる状態遷移のサポート

... lots more coming. 
● 日付・URL・タイトルからの検索
